Порядок перебора потоков идет сверху вниз. Их можно свободно перетаскивать потоки между собой, чтобы изменить порядор.

## Свойства потока

|Name|Default|Information|
|----|-------|-----------|
|Label|None|Имя потока|
|Type redirect|Http Redirect|Тип редиректа для перераспределения трафика|
|Input|None|Указываем ссылку, js, html код и другие значения характерные для выбранного Вами типа редиректа|
|Relation|&&|Все условия фильтра должны быть выполнены (**&&**) или любой (**\|\|**) из них|
|This is bot|False|Пометить поток, как поток для сбора ботов|
|Active|True|Активная ли группа в текущий момент|
|Logging|True|Сохранять подошедшие под условия переходы в текущем потоке в лог|

> **Предупреждение!** <br/>   Внимательно относитесь к пункту Relation <br />
**&&** - абсолютно все. Значит все условия фильтров должны подойти, чтобы сработал редирект по потоку. <br />
**||** - любой. При любом первом подходяшем фильтре, сработает редирект по потоку.

## Рекомендация

Первым потоком лучше всего стоит собирать ботов. При этом не забывайте помечать его как **This is bot**. Чтобы уменьшить нагрузку на Ваш сервер рекомендуется не логгировать этот поток.

> **P.S.** Не забывайте сохранять все свои изменения!
